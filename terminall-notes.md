
# summary of scrolling commands in NeoVim (Normal Mode):

  Scrolling Up
   * One line: k (cursor move) or Ctrl + y (viewport scroll)
   * Half screen: Ctrl + u
   * Full screen: Ctrl + b (Back)
   * Go to Top: gg

  Scrolling Down
   * One line: j (cursor move) or Ctrl + e (viewport scroll)
   * Half screen: Ctrl + d
   * Full screen: Ctrl + f (Forward)
   * Go to Bottom: G

  Other
   * Center Cursor: zz (centers the line the cursor is on in the window)
