# Lazygit Cheatsheet

## Navigation
- `h` / `l` — switch panels (left/right)
- `j` / `k` — move up/down in a panel
- `Enter` — expand/drill into item
- `Esc` — go back up / close

## Viewing Diffs
- Select a file in **Files** panel — diff shows on the right automatically
- Select a commit in **Commits** panel — full commit diff shows on the right
- `Enter` on a commit — opens its **file list**, then select a file to see its individual diff
- In file list: `j`/`k` to browse files, diff updates on the right

## Staging & Committing
- `Space` — stage/unstage file
- `a` — stage/unstage all
- `Enter` on a file — view hunks, `Space` to stage a hunk
- `v` in hunk view — select individual lines
- `c` — commit staged changes
- `A` — amend last commit

## Branches
- `n` — new branch
- `Space` — checkout branch
- `M` — merge into current branch
- `r` — rebase

## Other
- `p` / `P` — pull / push
- `z` — undo (reflog)
- `?` — full keybindings help
- `q` — quit
